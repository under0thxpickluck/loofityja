# RMTサイト 設計ドキュメント

**作成日:** 2026-04-14  
**参考サイト:** https://rmt.club/

---

## 概要

rmt.club を参考にした RMT（リアルマネートレード）サイトのフロントエンド実装。  
まずは見た目の再現を目的とし、将来のバックエンド機能追加（ユーザー登録・ログイン・出品）を見据えた構成とする。

---

## 技術スタック

- **フレームワーク:** Next.js 14（App Router）
- **スタイリング:** Tailwind CSS
- **言語:** TypeScript
- **カルーセル:** embla-carousel-react
- **データ:** ローカルJSONファイル（ダミーデータ）→ 将来APIに差し替え

---

## ページ構成

### 1. トップページ（`/`）

以下のセクションを上から順に配置する。

| セクション | 内容 |
|---|---|
| ヘッダー | ロゴ（RMTsite）、ゲーム名検索バー、ログイン/会員登録リンク |
| グローバルナビ | PCゲーム / モバイルゲーム / その他 / 全タイトル一覧 のリンク |
| カルーセル | プロモーションバナー（自動スライド、3〜4枚） |
| メインコンテンツ + サイドバー | 2カラムレイアウト（メイン: 左、サイドバー: 右） |
| └ ゲームランキング | モバイル・PCそれぞれトップ5のランキング表（件数付き） |
| └ 最新出品一覧 | 出品カードをグリッド表示（画像・タイトル・価格） |
| └ サイドバー | カテゴリナビ、ゲームタイトル検索入力欄 |
| フッター | 利用規約・プライバシーポリシー・お問い合わせ・コピーライト |

### 2. カテゴリ一覧ページ（`/category`）

- ヘッダー・フッターは共通コンポーネントを使用
- メインエリア: ゲームタイトルを「PCゲーム」「モバイルゲーム」「その他」の3カテゴリに分けてグリッド表示
- サイドバー: カテゴリ絞り込みリンク（すべて / PCゲーム / モバイルゲーム / その他）
- 各ゲームタイトルカードはクリック可能（将来的に出品一覧ページへ遷移）

## コンポーネント設計

```
app/
├── layout.tsx              # 共通レイアウト（Header, Footer を内包）
├── page.tsx                # トップページ
└── category/
    └── page.tsx            # カテゴリ一覧ページ

components/
├── Header.tsx              # ロゴ・検索バー・ログイン/登録リンク
├── Footer.tsx              # フッターリンク群
├── Carousel.tsx            # embla-carousel を使ったバナースライド
├── GameRankingTable.tsx    # モバイル/PCの2列ランキング表
├── ListingCard.tsx         # 出品1件分のカード（画像・タイトル・価格）
└── Sidebar.tsx             # カテゴリナビ・タイトル検索フォーム

data/
├── games.json              # ゲームタイトル一覧（id, name, category, listingCount）
└── listings.json           # 出品データ（id, gameId, title, price, imageUrl）
```

---

## デザイン仕様

- **カラー:** 白背景（`#ffffff`）、青アクセント（`#1a73e8`）、グレー補助（`#f5f5f5`, `#555`）
- **フォント:** システムフォント（sans-serif）
- **レイアウト:** 最大幅 1200px、中央寄せ
- **レスポンシブ:** モバイルではサイドバーを非表示、1カラムに切り替え

