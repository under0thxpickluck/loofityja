# LIFAI換金 お客様向けガイドページ — 設計仕様

## 概要

LIFAI換金サービスのお客様向け使い方マニュアルページを新規作成する。
申請フォーム（`/pc/lifai/sell`）とは独立した公開ページとして `/lifai/guide` に配置。
日本語のみ、ログイン不要で閲覧可能。

---

## ルート・ファイル構成

```
app/[locale]/lifai/guide/page.tsx   ← 新規作成（Server Component）
```

- `[locale]` ルーティングはそのまま使用するが、コンテンツは日本語ハードコード
- i18n翻訳キーは使用しない
- 既存の `/guide` ページのTailwindスタイルパターンを踏襲

---

## ページ構成

### 1. Hero セクション
- eyebrow ラベル（例：「LIFAI 換金ガイド」）
- メインタイトル（例：「EPをUSDTに換金する方法」）
- サブコピー（サービス概要を2〜3文）
- CTA ボタン：「今すぐ換金申請」→ `/pc/lifai/sell`

### 2. フロー概要（4ステップ ミニカード）
横3〜4列のカードで全体の流れを一覧表示。

| Step | タイトル |
|------|---------|
| 1 | アカウント登録・ログイン |
| 2 | プランとEP数量を選ぶ |
| 3 | 申請してEPを送金 |
| 4 | USDTを受取 |

### 3. 詳細ステップ（左右交互レイアウト）
`/guide` の `DetailSteps` コンポーネントと同じ構造（番号バッジ + タイトル + 説明 + インラインSVGイラスト）。

| Step | タイトル | 主な説明内容 |
|------|---------|------------|
| 1 | アカウント登録・ログイン | 未登録はサインアップ、既存ユーザーはログイン後に申請ページへ |
| 2 | プランを選ぶ | LIFAI内プランに合わせて選択。プラン別換金レート表（starter〜infra）を掲載。誤選択時は取引キャンセルの可能性あり |
| 3 | EP数量・受取ウォレットを入力 | 最低100EP、ネットワークはTRC20のみ、ウォレットアドレスは慎重に確認 |
| 4 | 申請してEPを送金 | 申請後に表示される運営ウォレットアドレスへEPを送金。申請後はなるべく早めに送金してください（システムが自動確認中） |
| 5 | USDT受取を確認 | EP着金確認後24〜72時間以内に指定ウォレットへUSDT送金 |

**プランレート表**（Step 2内に掲載）:

| プラン | EP換金レート |
|--------|------------|
| Starter | 0.25 円/EP |
| Builder | 0.2857 円/EP |
| Automation | 0.3333 円/EP |
| Core | 0.40 円/EP |
| Infra | 0.50 円/EP |

手数料は換金額の **8.5%**。

### 4. FAQ セクション（アコーディオン）

| # | 質問 | 回答骨子 |
|---|------|---------|
| 1 | 手数料はいくらですか？ | 換金額の8.5% |
| 2 | 最低何EPから換金できますか？ | 100EP以上 |
| 3 | 対応ネットワークは？ | TRC20のみ |
| 4 | EPはどのくらいで確認されますか？ | ご送金確認後、24時間〜72時間以内にUSDTを送金 |
| 5 | ウォレットアドレスを間違えたら？ | 送金後は取消不可。申請前に必ずご確認ください |
| 6 | プランはどう選べばいいですか？ | LIFAI内のプランに合わせてお選びください。プランの選択を誤った場合、換金レートの相違により取引のキャンセルとなる場合がございます |
| 7 | 申請後にEPを送らなかったらどうなる？ | タイムアウト後にキャンセル扱い。再度申請が必要です |
| 8 | 複数の申請を同時にできますか？ | 1件処理中は新規申請不可。完了後に次の申請が可能です |

### 5. CTA バナー（ページ最下部）
ダーク背景（`#0b1929`）のフルワイドバナー。「今すぐ換金申請する」ボタン → `/pc/lifai/sell`

## 実装方針

- **Server Component**のみ（`'use client'` 不要、インタラクションはFAQの`<details>`要素を使用）
- スタイル：既存 `/guide` ページのTailwindクラスを踏襲
- イラスト：シンプルなインラインSVG（新規画像ファイル不要）
- ナビゲーション：既存ヘッダーのリンクには追加しない（申請フォームからリンク、またはSEOで流入想定）

---

## 対象外

- 多言語対応（日本語のみ）
- ログイン認証（公開ページ）
- 既存ナビゲーションへの追加
