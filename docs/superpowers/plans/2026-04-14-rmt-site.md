# RMTサイト Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** rmt.club を参考にした RMT サイトのフロントエンド（トップページ + カテゴリ一覧ページ）を Next.js 14 で実装する。

**Architecture:** Next.js 14 App Router を使い、`app/layout.tsx` で Header/Footer を共通化する。各ページは Server Component として実装し、ダミーデータは `data/*.json` から読み込む。インタラクティブなコンポーネント（Carousel）のみ `"use client"` とする。

**Tech Stack:** Next.js 14, TypeScript, Tailwind CSS, embla-carousel-react, Jest, @testing-library/react

- [ ] **Step 1: Carousel を実装**

`components/Carousel.tsx`:

```typescript
'use client'

import useEmblaCarousel from 'embla-carousel-react'
import Autoplay from 'embla-carousel-autoplay'
import { useRef } from 'react'

const BANNERS = [
  { id: 1, text: '24時間365日リアルタイム取引対応', bg: 'bg-blue-600' },
  { id: 2, text: '初心者向け取引ガイド完備', bg: 'bg-blue-700' },
  { id: 3, text: '多様な決済方法に対応', bg: 'bg-blue-800' },
]

export default function Carousel() {
  const autoplay = useRef(Autoplay({ delay: 4000, stopOnInteraction: false }))
  const [emblaRef] = useEmblaCarousel({ loop: true }, [autoplay.current])

  return (
    <div className="overflow-hidden" ref={emblaRef}>
      <div className="flex">
        {BANNERS.map((banner) => (
          <div
            key={banner.id}
            className={`flex-none w-full ${banner.bg} text-white py-12 text-center text-lg font-bold`}
          >
            {banner.text}
          </div>
        ))}
      </div>
    </div>
  )
}
```

```typescript
// app/category/page.tsx
import Sidebar from '@/components/Sidebar'
import gamesData from '@/data/games.json'

const CATEGORY_LABELS: Record<string, string> = {
  pc: 'PCゲーム',
  mobile: 'モバイルゲーム',
  other: 'その他',
}

export default function CategoryPage({
  searchParams,
}: {
  searchParams: { type?: string }
}) {
  const activeType = searchParams.type

  const categories = activeType
    ? [activeType]
    : ['pc', 'mobile', 'other']

  const filteredGames = activeType
    ? gamesData.filter((g) => g.category === activeType)
    : gamesData

  return (
    <div className="max-w-screen-xl mx-auto px-4 py-6">
      <div className="flex gap-6">
        {/* メインコンテンツ */}
        <div className="flex-1 min-w-0">
          <h1 className="text-lg font-bold text-gray-800 mb-6">ゲームタイトル一覧</h1>

          {categories.map((cat) => {
            const games = filteredGames.filter((g) => g.category === cat)
            if (games.length === 0) return null
            return (
              <div key={cat} className="mb-8">
                <h2 className="text-sm font-bold text-blue-600 border-b border-blue-200 pb-1 mb-3">
                  {CATEGORY_LABELS[cat] ?? cat}
                </h2>
                <div className="grid grid-cols-2 sm:grid-cols-3 md:grid-cols-4 lg:grid-cols-5 gap-3">
                  {games.map((game) => (
                    <a
                      key={game.id}
                      href={`/game/${game.slug}`}
                      className="block bg-blue-50 hover:bg-blue-100 border border-blue-100 rounded px-3 py-2 text-sm text-center text-blue-700 hover:text-blue-900 transition-colors"
                    >
                      {game.name}
                    </a>
                  ))}
                </div>
              </div>
            )
          })}
        </div>

        {/* サイドバー */}
        <div className="hidden lg:block">
          <Sidebar />
        </div>
      </div>
    </div>
  )
}
```
